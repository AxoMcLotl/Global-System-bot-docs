## 🤖 About the Bot / Über den Bot

**Global System Bot** is a cross-server global chat bot for Discord that connects communities.
*   **Global Chat Relay:** Messages sent in the designated global channel are relayed instantly to all other connected servers.
*   **AutoMod & AI Moderation:** Integrated automated filters and AI-based image scanning ensure a clean and safe chat experience.
*   **Leveling System:** Users earn XP and level up by participating in the global chat.

**Global System Bot** ist ein Server-übergreifender Global-Chat-Bot für Discord, der Communities verbindet.
*   **Global-Chat-Weiterleitung:** Nachrichten, die im festgelegten Global-Kanal gesendet werden, werden sofort an alle anderen verbundenen Server weitergeleitet.
*   **AutoMod & KI-Moderation:** Integrierte automatische Filter und eine lokale KI-Bilderkennung sorgen für eine saubere und sichere Chat-Umgebung.
*   **Level-System:** Nutzer sammeln XP und steigen im Level auf, indem sie am globalen Chat teilnehmen.
